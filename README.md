# Titanic Survival Prediction

A beginner-friendly machine learning project built for Kaggle's Titanic: Machine Learning from Disaster competition.

## Project Overview

This project predicts whether a passenger survived the Titanic disaster using passenger details such as class, sex, number of siblings/spouses aboard, and number of parents/children aboard.

The model uses a Random Forest Classifier and generates a Kaggle-compatible submission file.

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Random Forest Classifier

## Dataset

Dataset used: Kaggle Titanic - Machine Learning from Disaster

Files:

- `train.csv`
- `test.csv`
- `gender_submission.csv`

## Features Used

- Passenger Class
- Sex
- Siblings/Spouses Aboard
- Parents/Children Aboard

## Model

Random Forest Classifier
